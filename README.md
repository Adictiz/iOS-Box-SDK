# Box iOS SDK

Last stable version is [1.0.1][archive]

---
## Requirements

- Xcode project must target iOS 6.0 or later
- Xcode project must link frameworks listed below :
	- `MobileCoreService.framework`
	- `UIKit.framework`
	- `SystemConfiguration.framework`
	- `webkit.framework`
